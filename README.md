# easyManage 学生管理系统

项目使用next.js 全栈TypeScript构成

## 启动项目

1. 下载依赖
   ```npm install```
2. 本地调试
   ```npm run dev```
3. 打包
   ```npm run build```
4. 启动服务器
   ```npm run start```

## 项目依赖

```
  1. tailwind
  2. prisma
```
